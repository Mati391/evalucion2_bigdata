Evaluación 2 - Big Data

Integrantes: Matías Sánchez, Jonathan Ávila, Yuliana Benitez

Este proyecto analiza datos de avistamientos de OVNIs en Estados Unidos utilizando herramientas de Google Cloud como BigQuery, Cloud Storage y Dataprep.

Resumen del Proyecto

  • Nombre del archivo: ufo_sighting.csv
  • Tamaño: 12.9 MB – 25.699 registros
  • Limpieza realizada en Cloud Dataprep
  • Consultas en BigQuery
  • Visualizaciones: gráfico de torta, gráfico de barras, mapa de calor por estado
